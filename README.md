# A2L language support

## Syntax highlighting
Basic syntax highlighting for ASAP2 MCD2 A2L files. See https://www.asam.net/standards/detail/mcd-2-mc/wiki/ for details.

## Code folding
Supports code folding between `/begin` and `/end` markers.
